## Style rules

- State the exercise. Then list any hints in a separate "Hint:" line.
- Difficulty tags: ★ (warm-up), ★★ (standard), ★★★ (extension).
- Every ★★★ exercise must reference a specific theorem in the chapter.
- If the exercise uses code, it must be runnable from the matching notebook in `notebooks/`.
